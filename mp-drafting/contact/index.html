Before You Send

What to send so we can respond accurately

Most delays on the front end come from unclear scope or missing inputs. If you want a faster, cleaner response, send the items below.

If a deadline is driving the request, say so plainly. Contractors often write: "The sooner the better as they want this installed ASAP." That context changes how we triage questions and next steps.

  • Architectural drawings (PDF, and DWG if available)
  • Specs that drive submittal requirements
  • Scope confirmation (your quote, a highlighted set, or a written scope statement)
  • Any known constraints (release dates, fabrication deadlines, phased areas)
  • The best point of contact for questions

Active Projects

Active project support

If you are already working with MP Drafting, use a subject line that makes routing obvious. Keep requests specific so revisions do not get lost in a long email chain.

Typical messages we receive include follow-ups like: "Hope your week was great man, just wanted to follow up on this project…" Clear identifiers help us answer without delays.

  • Put the Project Name in the subject line
  • Reference the sheet, elevation, grid line, or opening the comment applies to
  • Paste the comment in the email body, not only in a screenshot
  • If a live review is faster, say: "Please call, or send teams invite for date/time."

Engineering

Engineering questions

MP Drafting provides planning and shop drawings. Engineering is structural validation and must be performed by a Professional Engineer when required. MP can coordinate engineering workflows, but we do not position drafting as a replacement for a PE.

If you are unsure whether a job requires engineering, tell us what system and scope you are working with and what the review authority is asking for.

  • If you already have an engineer, we coordinate around that workflow.
  • If you need engineering support coordinated, note it in your request so it is routed correctly.
